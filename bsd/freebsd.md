# FreeBSD

## Ports

Searching ports:

```
cd /usr/ports
make index
make quicksearch name=fluxbox
```

Installation:

```
cd /usr/ports/x11-wm/fluxbox
make install
```

Updating:

```
portsnap fetch update
```
