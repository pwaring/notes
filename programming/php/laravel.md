# Laravel

## Requirements

 * PHP 5.6.4
 * OpenSSL extension
 * PDO extension
 * MBstring extension
 * Tokenizer extension

## Installing Homestead

Clone from GitHub:

```bash
git clone https://github.com/laravel/homestead.git ~/dev/homestead
```

Run initialisation script:

```bash
cd ~/dev/homestead
bash ./init.sh
```

Edit `~/.homestead/Homestead.yaml`.
